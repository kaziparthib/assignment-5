
 1.var,let and const
var: The oldest way to declare variables; it can be updated and redefined anywhere in its function.
let:The modern choice for variables that need to change; it is restricted to the specific block where it is defined.
const: Used for variables that should never change; once you assign a value, it cannot be updated or redefined.




 2.spread operator
The spread operator allows you to quickly copy elements from an array or object into a new one. It is commonly used to combine multiple arrays or add new properties to an object without changing the original.




   3.map(),filter(),forEach()
map(): Transforms every item in an array and returns a new array of the same length.

filter(): Checks every item against a condition and returns a new array containing only the items that pass.

forEach(): Simply loops through the array to perform an action for each item but does not return a new array.



 
 4.arrow function

An arrow function is a shorter, more compact way to write functions using the => syntax. It is popular because it requires less code and handles the this keyword more predictably than traditional functions.




 5.tamplate literals
Template literals allow you to create strings using backticks instead of quotes. They make it easy to insert variables directly into text using ${variable} and allow you to write strings across multiple lines easily.
